# Renewable Share Forecasting in Italy

This project forecasts Italy’s daily renewable electricity share using official Terna actual generation data.

## Data source

The raw data are downloaded from the Terna Data Portal:

https://dati.terna.it/en/download-center

Dataset: Actual Generation  
Years used: 2021–2025  
Primary sources: Thermal, Hydro, Geothermal, Photovoltaic, Wind, Self-consumption
